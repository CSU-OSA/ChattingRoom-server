# 信息码 解释文档

| MsgCode |      含义      |
|:-------:|:------------:|
|    0    |      成功      |
|    1    |    服务器异常     |
|    2    |   服务器拒绝响应    |
|   100   |   非法Method   |
|   101   |     参数不全     |
|   102   | 操作过于频繁，拒绝处理  |
|   200   |     非法昵称     |
|   201   |  用户信息认证不通过   |
|   202   |    昵称不可用     |
|   300   |    非法频道名     |
|   301   | 频道不存在或密钥认证失败 |
|   302   |    频道已存在     |
|   303   |   您被踢出了频道    |
|   400   |  服务器主动断开了链接  |
